# OpenWeatherMap API Documentation

Beginner-friendly API documentation for the OpenWeatherMap REST API, written as a docs-as-code portfolio project.

## Overview

This project documents selected endpoints from the OpenWeatherMap API. Each document covers the endpoint, request parameters, sample request, sample response, response fields, and error responses.

## Documentation Set

| Document | Description |
|----------|-------------|
| [Current Weather](docs/current-weather-endpoint.md) | Get real-time weather data for a city |
| [5 Day Forecast](docs/five-day-forecast-endpoint.md) | Get weather forecast every 3 hours for 5 days |
| [Air Pollution](docs/air-pollution-endpoint.md) | Get air quality index for a location |
| [Geocoding](docs/geocoding-endpoint.md) | Convert city names to coordinates |

## Tools Used

## Who This Is For

- Beginners learning how to read and use REST APIs
- Aspiring technical writers building a docs-as-code portfolio
- Anyone who wants clear, plain-English API documentation

## Notes

This project uses the free tier of the OpenWeatherMap API.
All sample responses are real responses retrieved during testing.
